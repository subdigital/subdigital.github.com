Workshop on Continuous Integration This Saturday

Sep 16th, 2009

This is pretty last-minute, but we’re putting together a workshop on Continuous Integration.

We’ll cover the principles and the tools, and get a working solution up in front of the group.

 

 

Some topics we’re going to cover:

  • Source control
    • svn
    • git
    • perhaps TFS (since many of us are forced to work in with this)
  • Build frameworks
    • NAnt
    • Rake
    • others? (PSake, Phantom)
  • CI Tools
    • TeamCity
    • CruiseControl.NET
    • Hudson
  • Testing
    • NUnit / MbUnit

 

When is it?

This Saturday, September 19, 2009.  9:00am-2:00pm

Where is it?

It will be at the Microsoft office, located at:

2000 W Sam Houston Pkwy S #350
Houston, TX 77042-3624

 

We’re trying to get pizza for the event, so if you’re going, please let a comment so that we get an accurate head count.
